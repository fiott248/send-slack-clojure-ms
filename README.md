# slackhook-app

A simple slack hook wrapper using clojure.


## Installation

Install Lein FIRST

lein uberjar  

docker build -t fiott248/slack-wrapper .

docker run -p 8090:8090 fiott248/slack-wrapper

## Usage

http://127.0.0.1:8090/slack
